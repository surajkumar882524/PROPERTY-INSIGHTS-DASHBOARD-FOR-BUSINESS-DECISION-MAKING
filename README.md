# PROPERTY-INSIGHTS-DASHBOARD-FOR-BUSINESS-DECISION-MAKING
A data-driven dashboard analyzing global property revenue, expenses, and profits to support strategic business decisions and identify top-performing agents and markets   Data cleaning, DAX formulas, data modeling, KPIs, map visualization, slicers, and interactive charts (line, bar, and card visuals)..Glimps Properties Sales Dashboard: An interactive Power BI dashboard tracking property sales performance from 2022-2024. Key visuals include revenue by country (top: China, Italy, Philippines), agent leaderboards (e.g., Olga Glimps, Tamara Moore), quarterly/monthly trends, and top-selling properties with profit summaries (~9M profits, 26 units sold).


# DASHBOARD HIGHLIGHTS
1 Revenue by Country: World map and table showing top-performing markets like Brazil, Sweden, and France.

2 Monthly Revenue Trends: Bar chart visualizing monthly fluctuations, with July flagged for low performance and October for peak sales.

3 Quarterly Expense Breakdown: Clear visualization of spending across Q1–Q4, with Q4 as the highest.

4 Top Contributors: Table listing individual revenue contributions, led by Benjamin Moore (5M).

5 Property Visuals: Embedded images of top-selling and cozy properties to enhance storytelling.

# PROJECT DEVELOPMENT PROCESS

1 Data Collection & Preparation

- Gathered property sales, revenue, and expense data from Excel/CSV sources.
- Cleaned and formatted datasets for consistency (e.g., standardizing country names, ensuring date formats).
- Verified totals (Revenue, Expenses, Properties Sold) for accuracy before importing.
  
2 Data Import into Power BI
- Loaded datasets into Power BI Desktop.
- Established relationships between tables.
- Used Power Query for transformations and filtering.
  
3 Data Modeling
- Created calculated columns and measures using DAX (e.g., Net Profit = Revenue – Expenses, Average Revenue per Property).
- Built hierarchies (Year → Quarter → Month) for drill-down analysis.
- Ensured proper aggregation for summary metrics.
 
4 Dashboard Design
- Designed multiple visuals:
-  Map for Revenue by Country
-  Bar chart for Monthly Revenue Trends
-  Line/column chart for Quarterly Expenses
-  Table for Revenue by Individual
-  Added images of properties for storytelling and user engagement.
-  Applied conditional formatting (e.g., highlighting July as low revenue, October as peak).
 
5 Insights & Business Questions
- Connected visuals to answer key business questions (Revenue contribution, Expense efficiency, Seasonal performance, Sales productivity).
- Highlighted actionable insights like growth potential in moderate-revenue countries and expense management in Q4.
  
6 Finalization & Publishing
- Created summary cards for Revenue, Expenses, Profit, Properties Sold.
- Polished layout with consistent colors, fonts, and alignment.
- Published dashboard for portfolio demonstration and GitHub documentation.
